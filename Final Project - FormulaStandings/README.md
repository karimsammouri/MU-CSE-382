<h1>FormulaStandings</h1>

<h3>A Formula One® Companion Mobile App</h3>

<p>Displays the latest season's race schedule, race results, driver standings, constructor (team) standings, driver information, constructor information, and latest news. Built using C#, XAML, Microsoft's Xamarin.Forms cross-platform UI framework, and Visual Studio. Makes use of a local database, <i>ListViews</i>, and user preferences. Calls on the Ergast Developer API web service for the latest Formula One® race, driver, and constructor data. Abiding by the terms and conditions of the Ergast web service, this project is intended for personal, non-commercial purposes. It fulfills the final project requirements of Miami University's CSE 382: Mobile Application Development for the Fall of 2022. This project, excluding the Ergast web service, is wholly designed and implemented by me with all rights reserved. The following samples are based on the Xcode iPhone 14 (iOS 16.2) simulator and the 2022 Formula One® season.</p>
